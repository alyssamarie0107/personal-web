iApply

iApply is a job application management and preparation tool. It is different than other apps because we not only help you manage your applications, but also provide insights to help you put your best resume forward. Much of initial resume parsing today is being done via computers rather than human recruiters, so tailoring your resume to the job requirements is increasingly important. Our mission is to ease the burden of tracking your applications so time can be better spent preparing for the opportunity itself.

Features of iApply:

  • Allow user to create applications with company, notes, due date, location, job requirments, and posting url (uses Google Place API)
  • Users can scan their resume and compare the job requirments of each company to their resume (uses OCR and copy and pasting of job requirments)
  • Use the To Do list to keep track of upcoming deadlines, coding challenges, and interview dates (integrated with Apple Calendar)

Card Bank

Built for my ECS 189E course in Winter 2021. The app is a wallet, which can be broken down into three parts: login, wallet, and transaction.

For the login part, Card Bank uses phone number authentication. It takes in a phone number from the user and sends a verification code to the phone number by text message. Card Bank then prompts the user to enter the code and verify user's identity (it is assumed that the user will use a US phone number only).

For the wallet part, Card Bank displays the user's wallet, which has a table of accounts and the information of each account.

For the transaction part, Card Bank allows the user to make transactions between the accounts in the wallet.

Pomocon

Pomocon is a website that acts as a Pomodoro timer. It allows users to select between a 25 minute work interval or 5/10/15 minute breaks. Pomocon displays the time interval the user selects and shows the exact time the interval will be done. The time interval is displayed in a aesthetically pleasing circle, which depletes with time accordingly. When the timer is up, an audio notification plays to let the user know that they completed a Pomodoro or that their break is over. Pomocon also keeps track of the user's total time for their session on Pomocon. The user will be able to see their work time, break time, total time, completed Pomodoros, and efficiency.
